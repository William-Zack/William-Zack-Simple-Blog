## 一个简单的博客网页

"Web前端开发技术"课程的大作业。

项目已经制作完成，同时已经上传到[GitHub](https://github.com/William-Zack/William-Zack-Simple-Blog)。

因为是"Web前端开发技术"的大作业，所以对于用户数据，此网页仅存储在localStorage与sessionStorage中。

就这样。

✅登录页面以及对应检测逻辑

✅完整的顶部栏与底部栏（头像更换，搜索栏，退出登录等）
